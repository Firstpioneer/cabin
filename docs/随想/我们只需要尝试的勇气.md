很多时候，困难并不来自于外部，而是我们自己。

想参加活动，但因犹豫而裹足不前。

想线下面基，却担心“彼此不认识会尴尬”而选择逃避。

遇到flipped的女孩，更是在臆想之后自我安慰说“大概是多巴胺的躁动”。

在不知不觉中我们给自己套上了重重的枷锁。

让我们始终自我怀疑，举足不前。


但实际情况呢，很多时候，当我们迈出脚步后，才会发现所有的困难都是由自己无限放大。


学校工作室面试，要求新生有前后端基础优先。犹豫很久，然后疯狂补进度，结果面试时，只talking了未来规划和学习计划。

实验室面试，说有笔试和面试，担心通过不了，做了很长时间的准备和学习，结果面试时，简单conversation完，直接让我跟着学长学习。

想学习SGLang架构，巧合下看到要组建SGL社区，犹豫了很久才鼓起勇气联系组建人。结果交流非常愉快，给了我很多学习相关架构的指导。

想报名Bohack，害怕自己过不了筛选，准备修改了很久的简历和博客，结果凌晨2点报名后，当天晚上就联系我有没有意向参与前端开发。


很多时候，我们总把事情想得太难，把本可争取的机会归咎于外因，却忘了“尝试”本身就是最简单的钥匙。

回想每一次的突破，都伴随着漫长的心理挣扎：工作室报名前的忍耐、实验室报名时的忐忑、加入SGLang社区时的不安、黑客松报名时的拖延。

一路上，我总在给自己负面的心理暗示：“这门课好难，我肯定学不会”，“这个课题组的方向我完全没接触过，老师肯定不会要我”。

我们亲手划定了一个舒适区，把“别人做不到我也做不到”，“我没接触过就一定比不过别人”当作信条，习惯于放大困难，却不愿立刻行动；宁愿安于现状，也不敢踏出舒适区。

但实际上，我们所缺少的只是迈出第一步的勇气。不去尝试，永远也不会知道。

我们只需要尝试的勇气。
