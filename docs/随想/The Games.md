# 浅议原神相关

最近玩原神的时候，突然感到十分的焦虑。因为游戏一直以来对我来说是消磨时间，获得乐趣的工具。以前玩原神的时候从来没有产生过在当下社会由于竞争而产生的焦虑。遂想记之，以析其因。

先说明以下本人的情况，零氪党，枫丹纳塔的角色一个没抽。

一切的起源可能在于我看了唔得比的抽卡分析视频而衍生的。

1，在看了这个视频的前一部分，我便也清晰的意识到了原神很失衡的一个现象：数值失衡。纳塔的角色过于强势，为了带给玩家抽取新卡的动力，因此将深渊，活动的难度提高进而让玩家从实际操作中感受到新角色的强势，一方面深渊怪物的血量得到进一步的提升，另一方面也增加了独属纳塔的次元盾。非纳塔角色破盾很是困难。在枫丹版本的时候，荒芒属性便初具雏形，本土角色打本土怪物有优势这无可厚非。但在纳塔，当怪物必须先通过重击或普攻要先把次元盾破掉时才能消耗敌人血量时，无纳塔角色在配队里便会非常难受。在枫丹版本其实还好，虽然出了很超标的龙王，仆人 ，但老角色，老配队在深渊其实还有一席之地。但随着纳塔版本更多更超标角色的出现，火神，希诺宁，玛拉妮，基尼齐等等，整体角色的强度在提升，随之而来便意味着：老角色老配队已经不行了。细想一下，蒙德似乎只有班尼特出场率还高一些。零氪党的零命雷神国家队最高也只能打15w，从开始玩原神便一直在玩的小提队，伤害也不过是重击3w，以前还能打通深渊。现在连深渊的11层第三间都打不满了。雷神草神似乎也在退出玩家的视野中心。

引爆这次失衡的也是这次的爬塔活动，不同于之前的任何挑战活动，这次的活动结果是以爬塔数作为比较标准，这个活动设计的其实是很好的，能在短短2分钟内就能衡量出一个队伍的爆发输出，但简单的层数也让对比显得更加鲜明，每一层的通过都是一个大阶段的突破。以往的活动是以积分衡量：“我打了100w的积分”，“我打了80w”，以往的活动并没有使对比变得尖锐，一方面积分没有上限，另一方面很大的数额使对比的冲击感也没有那么强烈，“你打了那么多，我打的也不少”，“我的手法不太连贯，多大两次我也能凹出100w”。而现在的30s就将差距从另一方面扩大，你会发现不论你怎么挑战，很难实现一层的跨越，活动的最低配也是龙芙队。即便是满命雷神，也对不过纳塔的新三体人。

2，我感到焦虑主要有两方面，

首先是看上面那个视频带给我的，不同与以往很全面很详细面向大多数用户的详实的角色培养攻略，这个抽卡分析动不动就是2命龙，满命仆人的，让我感觉这个视频的受众根本不是平民玩家，仿佛让我看到“天宫”一角，大家似乎都在追逐伤害有多高，几命的配对更强，除了氪佬和屯屯鼠谁手上会有如此多的抽数？根本没有这样本钱的我,只能默默享受着痛苦。就好比在学校里，老师经常讲某某某只要这道题没错就能怎么怎么样，并没有直接对比，但恰恰是这类事实的陈述所蕴含的隐形的对比，也会产生不小的焦虑。

其次就是身边好友的对比。很鲜明的就是抽有龙王，火神的朋友，打出的层数就是比你费尽心思苦练的队伍高。面对这样的请况感到的也很是无奈。

好吧，发癫之后，也是恢复了冷静。因为我突然意识到自己的游戏追求不同，似乎我是一个xp党，就算角色很弱，但也想抽了放进图鉴里。原神吸引我的还是剧情，以及每日的可有可无的消遣。而另外一些人追求的可能就是练度，就是整体的强度。道不同，不相为谋。基本没有什么对比的意义。

人生亦是如此。
