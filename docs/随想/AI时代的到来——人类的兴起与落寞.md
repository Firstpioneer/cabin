# **AI时代的到来——人类的兴起与落寞？**

## ——一个渺小者所感

1943年，McCulloch和Pitts提出人工神经元模型，奠定了神经网络的理论基础。  
1950年，Alan Turing发表论文《计算机器与智能》，提出图灵测试，成为判断机器是否具有智能的标准。  
1956年，达特茅斯会议召开，John McCarthy(约翰·麦卡锡）首次提出““人工智能”这一术语，标志着AI领域的正式诞生。  
1980年，机器学习开始兴起，统计学习方法如支持向量机（SVM），决策树等被广泛研究。  
2006年，深度学习技术开始受到关注，卷积神经网络（CNN）和循环神经网络（RNN）等架构被广泛研究。  
2012年，AlexNet在ImageNet竞赛中取得优异成绩，开启了深度学习在计算机视觉领域的广泛应用。  
2016年：谷歌的AlphaGo战胜围棋世界冠军李世石，展示了AI在复杂策略游戏中的强大能力。  
2020年，OpenAI发布了GPT-3，标志着大语言模型（LLM）时代的到来。  
2022年，ChatGPT发布，全球惊叹于GPT的回答，ChatGPT的爆火标志着人工智能正逐渐走进人们的日常生活中，AI对人类的影响也会越来越大。之后各类大语言模型层出不穷。  
2025年，中国deepseek推理模型爆火。

在高中学习期间，虽成绩稀里糊涂，但仍很是困惑于所学的物理，数学有何用？现实中的环境是那么的复杂，这些简简单单的模型是怎么实现模拟的。高考完后，也没有对世界形成一个很好的认识，稀里糊涂的选择了热门的计算机行业，一脸茫然的进入了这个不到一个世纪前才刚刚兴起的学科。  
在大学里，学着课堂的基础知识，虽然惊叹于计算机系统的巧妙设计，但仍是困惑于这个系统是如何实现现在我所敲的电脑的近乎全面的功能。  
**有时候真的会感觉自己是生活在信息时代的低能儿，无法想象先辈们是如何开创出来的。**
